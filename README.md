# SkillSwap 🚀

A full-stack web application where users can exchange skills with each other.

## 🔧 Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.IO

## ✨ Features

- User Authentication (JWT)
- Skill Matching System
- Swap Requests (Accept/Reject)
- Real-time Chat
- Video Call (WebRTC)
- Premium Subscription System

## 📁 Project Structure

frontend/ → React UI  
backend/ → Express API  

## 🚀 How to Run

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm start
